# AI_For_Marketing

Repository of example applications for teaching AI in marketing practice and research.

## Overview

This repository contains two marketing applications that demonstrate practical applications of AI and generative AI in marketing:

### 1. AI for Image Classification

A comprehensive application for image classification featuring:
- **Two Datasets**: Multiple datasets for training and evaluation
- **Vision Transformers**: Scripts and notebooks for implementing state-of-the-art vision transformer models
- **Vision Language Models**: Tools for leveraging multimodal models that combine vision and language understanding

Use cases include product categorization, visual content analysis, and automated image tagging for marketing campaigns.

### 2. GenAI for Influencer Marketing

An application focused on applying generative AI to influencer marketing, including:
- **Example Images**: Sample datasets and visual assets for demonstration
- **Analysis Tools**: Scripts for evaluating and selecting influencer content
- **Integration Examples**: Practical implementations for marketing workflows

## Execution Environment

All notebooks and scripts can be executed in two ways:

1. **Google Colab**: Run the notebooks directly in your browser without any local setup requirements
2. **Local Python Installation**: Run the code on your local machine with a Python environment and required dependencies

Choose the environment that best suits your workflow and available resources.
